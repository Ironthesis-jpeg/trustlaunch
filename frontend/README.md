TrustLaunch Frontend

This folder contains the frontend application for TrustLaunch.

Purpose

The frontend allows users to:

- Launch tokens
- Explore projects
- View founder profiles
- View Trust Scores
- Connect wallets
- Interact with the TrustLaunch platform

Planned Features

Homepage

- Hero section
- TrustLaunch statistics
- Launch types
- Top projects

Launch Page

- Meme Launch
- Safe Launch
- Pro Launch

Explore Page

- Browse projects
- Filter by Trust Score
- View Fair Launch projects

Founder Profiles

- Reputation Score
- Launch History
- Badges

Wallet Integration

- Base Network
- WalletConnect

Future Features
